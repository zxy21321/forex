MT4 – Häufig gestellte Fragen

Welche Plattformen bietet Axi an?

Axi bietet den Gratis-Download und die Installation der führenden Handelsplattform MetaTrader 4 (MT4) für die beliebtesten Geräte und Betriebssysteme wie Windows, Mac, iPhone/iPad und Android an sowie die browserbasierte WebTrader-Plattform.

Weitere Informationen zu unseren Plattformen für Privatkunden erhalten Sie auf der Seite Handelsplattformen. Wenn Sie sich für institutionelle Plattformen interessieren, besuchen Sie bitte die Website AxiPrime.

Wo kann ich die MT4-Handelsplattform herunterladen?

Wenn Sie einen Antrag auf Eröffnung eines Live- oder Demo-Kontos gestellt und ein Konto angelegt haben, erhalten Sie einen Link zugeschickt, mit dem Sie die Plattform auf Ihrem bevorzugten Computer oder Gerät herunterladen können.

Wenn Sie bereits ein Live-Konto eröffnet haben und die MT4-Plattform herunterladen, aktualisieren oder neu installieren möchten, finden Sie die neueste Version auf der Website von Axi über das Kundenportal. Navigieren Sie dazu zur Anmeldungsseite für Kunden im Hauptmenü und wählen Sie „MT4 herunterladen“. Sie können auch das Kundenerfahrungsteam per Telefon, E-Mail oder Live-Chat kontaktieren, um von den Mitarbeitern einen direkten Download-Link zu erhalten.

Wie kann ich meinen Handelsverlauf einsehen?

Um die Wertentwicklung vergangener Transaktionen einzusehen, rufen Sie die Historiendatenbank mit folgenden Schritten auf: 

  1. Starten Sie die MT4-Plattform
  2. Klicken Sie auf „Extras“ in der Navigationsleiste
  3. Wählen Sie „Historiendatenbank“ aus dem Dropdown-Menü

Welche Version von MT4 benutze ich?

Um die Versionsnummer Ihrer MT4-Plattform herauszufinden, starten Sie die Plattform und wählen Sie in der Menüleiste den Punkt Hilfe. Wählen Sie im Dropdown-Menü „Über“. Im folgenden Fenster erhalten Sie Informationen zur Versionsnummer sowie das Datum der letzten Aktualisierung.

Wie öffne ich einen Chart in MT4?

Es gibt zwei Möglichkeiten, einen Chart zu öffnen:

  • Klicken Sie mit der rechten Maustaste auf das gewünschte Symbol im Fenster „Marktübersicht“. Wählen Sie im Dropdown-Menü „Neues Chartfenster“
  • Klicken Sie auf die Schaltfläche „+“ in der Hauptsymbolleiste (siehe unten) und wählen Sie dann die gewünschte Währung im Dropdown-Menü

Wie ändere ich den Charttyp?
